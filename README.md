# jvcrii-2.0
Django002

Nombre del proyecto
jvcrii-2.0

Descripción
Proyecto sencillo creado con Django para aprendizaje.

Requisitos
Python 3.12
Git

Pasos para ejecutar el proyecto por primera vez

Clonar el repositorio desde GitHub

Entrar a la carpeta del proyecto

Crear un entorno virtual con Python

Activar el entorno virtual

Instalar Django

Ejecutar las migraciones iniciales

Levantar el servidor de desarrollo

Abrir el proyecto en el navegador en la dirección local

Cómo fue creado el proyecto

Se creó un proyecto Django usando django-admin

Se creó una aplicación Django usando manage.py

Se registró la app en settings

Se configuraron las URLs del proyecto y de la app

Se creó una vista básica para pruebas

Notas
Los archivos generados automáticamente por Python, la base de datos local y el entorno virtual no se suben al repositorio.
